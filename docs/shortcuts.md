# Atajos de teclado

## Modo Pregunta Respuesta

| Tecla | Acción |
|-------|--------|
| `1-5` | Seleccionar opción |
| `Space` / `Enter` | Siguiente pregunta (solo si ya respondiste) |

## Modo Examen

| Tecla | Acción |
|-------|--------|
| `1-5` | Seleccionar opción |
| `←` `→` | Navegar a pregunta anterior/siguiente |
| `Space` / `Enter` | Ir a siguiente pregunta |
| `Ctrl+Enter` | Abrir diálogo de confirmación "Entregar examen" |

## En ambos modos

- Los atajos se ignoran si el cursor está en un campo de texto (`INPUT` / `TEXTAREA`)
- Los atajos no funcionan durante el diálogo de confirmación ni en la revisión final
